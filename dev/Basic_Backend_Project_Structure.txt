Базовая структура приложения

hosteypic				    # Корневая папка проекта
└ .flaskenv			  	# Переменные среды приложения (не уязвимая информация)
└ .gitignore		  	# Неотслежевыемые файлы гитом
└ config.env			  # *Перменные среды приложения (уязвимая информация)
└ app.db				    # *БД на время разработки
└ dockerfile
└ hosteypic.py			# Точка входа в приложение (через flask run)
└ tests.py				  # Юнит-тесты
└ requirements.txt	# Требуемые для развертывания и разработки пакеты pip
└ app					      # Шаблоны, модули
  └ auth				    # Модули, необходимые для работы аутентификации, регистрации, восстановлений и подтверждений
    └ routes.py			# Пути веб-приложения и привязанные к ним функции
	  └ ...
  └ errors				  # Модули (хендлеры) для обработки ошибок
  └ main				    # Модули для основного функционала сервиса
    └ routes.py			# Пути веб-приложения и привязанные к ним функции
	└ ...
  └ static				  # Статичные файлы приложения (стили, скрипты и картинки юзеров**)
    └ userfiles			# **Файлы пользователей
	  └ ...				    # Скрипты и стили
  └ templates		
    └ auth
	  └ email
	  └ errors
	  └ ...
  └ ...					    # Модели БД, модули дополнительные основного функционала
└ logs					    # *Логи приложения
└ migrations			  # Миграции БД
└ venv					    # Виртуальное окружение Python

* НЕ ВКЛЮЧАТЬ В СИСТЕМУ КОНТРОЛЯ ВЕРСИЙ (прописать в .gitignore)
** Пока что достоверно не известно...
